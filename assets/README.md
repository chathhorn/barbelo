# Application visual assets

The files in this directory are project assets distributed under Barbelo's
root MIT license. They are self-contained: the SVGs contain no linked images,
scripts, external fonts, or other network dependencies.

- `favicon.svg` is the ouroboros application mark.
- `pleroma.svg` is the alternate monad/compass mark, extracted from the
  original inline header artwork so the two header variants use the same
  image-loading path.
- `collie-01.svg` through `collie-20.svg` are project-created, geometric
  variants of the Pair Improvement Report coach portrait. Variants 1–10 were
  introduced in commit `af080b9`; variants 11–20 and their additional
  expressions were introduced in commit `1ab6237`.
- `barbelo.css` is the application stylesheet.

Bridge Simulator has a separate asset pack and creation record in
`packages/bridge-simulator/assets/ATTRIBUTIONS.md`.
