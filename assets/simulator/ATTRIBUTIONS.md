# Bridge Simulator visual assets

All visual files in this directory are project-original assets created for
Barbelo's Bridge Simulator. No third-party artwork is included in this pack.

## Creation record

- Created: 2026-07-12.
- Method: hand-authored SVG markup generated by OpenAI Codex from the written
  Bridge Simulator brief, using integer-coordinate rectangles, polygons,
  paths, and flat colors.
- Raster/image model use: none.
- External fonts, linked images, scripts, filters, and copied vector paths:
  none.
- Existing Barbelo Coach photographs, avatars, and SVGs were not used as
  visual references, traced, sampled, color-picked, or transformed. The Coach
  pose set was designed from scratch from the generic prompt “full-body Border
  Collie bridge coach.”
- Doom/WAD/source-port content: none. No Doom geometry, textures, sprites,
  UI, logos, characters, sounds, or converted data were consulted or used.
- Validation: each SVG was parsed with `xmllint` and raster-previewed locally
  with librsvg to check transparency and silhouette readability.

These originals are intended to be distributed under the repository's MIT
license. The upstream Three.js engine is vendored separately under
`vendor/three/` and retains its own MIT license, exact release record, source
URLs, and hashes there.

## Shared visual specification

- Font-free and self-contained.
- `shape-rendering="crispEdges"` with integer-coordinate geometry.
- Flat bridge-club palette: ink/navy, paper cream, felt green, score blue,
  review-slip gold, and restrained penalty red.
- Transparent backgrounds for billboard actors, projectiles, effects, and
  pickups; opaque backgrounds only for surface textures.
- Informational `<metadata>` records suggested world scale, anchor, role, or
  atlas cell layout. These values are not a standardized SVG schema.

For a pixel-authentic WebGL presentation, rasterize at the intrinsic SVG
dimensions and use nearest-neighbor minification/magnification. Do not smooth
or resize the source into a larger baked bitmap.
